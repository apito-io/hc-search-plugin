# hc-search-plugin

Elasticsearch and Meilisearch integration for the Apito API platform.

## Features

- Full-text search
- Elasticsearch support
- Meilisearch support
- Faceted search
- Search analytics

## Installation

```bash
go build -o hc-search-plugin .
```
